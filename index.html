Outlier Introduction

  • 2016/09/08

Introduction

Outlier definitions

  • A data point which is significantly different from the remaining data.
  • “An outlier is an observation which deviates so much from the other observations as to arous suspicions that it was generated by a different mechanism.” ——Hawkins
  • abnormalities, discordants, deviants or anomalies.

异常检测的应用

异常通常包含一些有关系统异常特征的有用信息以及实体，这类信息反映了数据产生的过程。通过这些异常特征可以通过一些应用进行检测，比如：

  • 入侵检测系统

    网络入侵等。

  • 信用卡诈骗

    分析异常的购买记录，比如在不确定的地点无节制的购买商品。

  • 感兴趣的传感器事件(Interesting Sensor Events)

    传感器状态突然的改变能够检测出一些问题。比如跌倒。

  • 医疗诊断

    通过一些检测技术，比如医疗成像，来发现特定的异常。

  • 法律实施

    从不正常的行为模式中发现违法行为

  • 地球科学

    通过空间数据（气候环境，变化以及地表覆盖等）发现潜伏的人群或者预测气候变化等。

在这些应用中，异常是通过正常的数据挖掘的，他们的输出结果往往是一个分数（Score, 对是否为异常点的可信度打分）或者一个标签(label，是否为异常)。但输出分数的方式往往可以通过给定的阈值(threshold)来更改为标签的形式。一般而言标签是最终的决策形式，但所提供的信息却相对较少。
